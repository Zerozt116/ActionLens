# 视频行为数据集调研

更新时间：2026-06-08

本文件记录适合“视频中谁在什么时间做了什么”任务的数据集候选。当前项目优先关注喝水、打电话、玩手机、使用设备等人-物交互行为。

## 候选数据集

| 数据集 | 适合用途 | 与本项目关系 | 链接 |
| --- | --- | --- | --- |
| NTU RGB+D / NTU RGB+D 120 | 喝水、打电话、玩手机等固定动作分类；包含 RGB、深度、骨架等模态 | 很适合验证喝水、打电话规则和姿态特征，但多为受控场景，不是监控视角 | https://rose1.ntu.edu.sg/dataset/actionRecognition/ |
| AVA Actions | 人级别时空动作检测，含人员框、动作标签和 person_id | 最贴近“谁在什么时间做什么”的任务定义，适合研究时空动作定位。已新增 `docs/AVA.md` 和 `scripts/prepare_ava.py` | https://research.google.com/ava/index.html |
| Charades | 室内日常活动视频，动作和物体组合丰富 | 适合室内行为理解、人-物交互语义和时间段动作定位；已新增 `docs/Charades.md` 和 `scripts/prepare_charades.py` | https://prior.allenai.org/projects/charades |
| EPIC-KITCHENS | 第一视角厨房人-物交互，动作和物体标注细 | 适合学习 cup、bottle 等细粒度物体交互，但视角不是监控/第三人称 | https://epic-kitchens.github.io/ |
| MEVA | 多摄像头、多人物、真实环境活动检测 | 适合后续研究监控场景、多视角活动检测和人员重识别 | https://mevadata.org/ |
| UCF101 | 通用动作识别基准 | 可作基础模型预训练或教学验证，但不适合精确人员级时空定位 | https://www.crcv.ucf.edu/research/data-sets/ucf101/ |
| ActivityNet | 长视频时间动作定位 | 适合研究行为开始/结束时间定位，但粒度偏视频片段级 | https://activity-net.org/ |
| STAIR Actions | 100 类日常动作短视频 | 可补充日常行为样例，适合动作分类预训练或对比 | https://actions.stair.center/ |

## 当前优先级

1. NTU RGB+D 120：优先用于喝水、打电话、玩手机动作样例。
2. Charades：用于室内日常行为泛化和时间段动作定位，先做小批量 pilot。
3. AVA：用于人员级时空动作检测路线参考，但视频下载受 YouTube 可用性影响较大。
4. MEVA：用于后续监控场景、多视角活动检测和人员重识别。
5. EPIC-KITCHENS：用于人-物交互和厨房细粒度物体交互参考。

## AVA Actions 重点信息

官方页面：https://research.google.com/ava/index.html

下载页面：https://research.google.com/ava/download.html

要点：

- AVA v2.2 包含 430 个视频。
- 训练集 235 个视频，验证集 64 个视频，测试集 131 个视频。
- 每个视频有 15 分钟片段被标注。
- 标注间隔为 1 秒。
- 标注包含人员框、动作 ID 和 person ID。
- 同一人可有多个动作标签。
- 数据采用 CC BY 4.0 许可。

与本项目相关的动作：

- `15`: answer phone -> `talking_on_phone`
- `27`: drink -> `drinking_water`
- `57`: text on/look at a cellphone -> `using_phone`

已新增：

- `docs/AVA.md`
- `scripts/prepare_ava.py`
- `tests/test_prepare_ava.py`

## Charades 重点信息

官方页面：https://prior.allenai.org/projects/charades

要点：

- 9848 个室内日常活动视频。
- 157 类动作。
- 约 66,500 个动作时间段标注。
- 标注包含视频 ID、场景、质量、脚本、描述、动作开始/结束时间等。
- 标注不包含人员框和 person ID，因此更适合时间段行为定位，不适合直接做人员框 IoU 评估。
- 数据许可为非商业研究用途，不能向第三方再分发。

## NTU RGB+D 120 重点信息

官方页面：https://rose1.ntu.edu.sg/dataset/actionRecognition/

官方说明要点：

- NTU RGB+D 包含 60 类动作、56,880 个样本。
- NTU RGB+D 120 扩展到 120 类动作、114,480 个样本。
- 每个样本包含 RGB 视频、深度图序列、3D 骨架数据、红外视频。
- RGB 视频分辨率为 1920x1080。
- 每个样本由 3 台 Kinect V2 摄像机同步采集。
- 数据集仅限学术研究和非商业用途。
- 下载需要注册账号、提交申请表并接受 Release Agreement。
- 完整 NTU RGB+D 120 全模态约 2.3TB；RGB 视频约 136GB + 124GB。

与当前项目直接相关的动作类别：

| 动作编号 | 英文标签 | 中文含义 | 当前用途 |
| --- | --- | --- | --- |
| A001 | drink water | 喝水 | 验证 `drinking_water` 规则 |
| A028 | phone call | 打电话 | 验证 `talking_on_phone` 规则 |
| A029 | play with phone/tablet | 玩手机/平板 | 后续扩展手机相关行为 |
